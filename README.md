# Contatooh

A simple contact manager implemented MEAN Stack.<br/>
This projects was based in "MEAN Book".<br/>

## Dependencies

``NodeJS``<br/>
``Bower``

## Persistence

``MongoDB``

## Running (locally)

``export MONGO_URI=mongodb://127.0.0.1/contatooh``<br/>
``npm install``<br/>
``npm run build``<br/>
``npm start``<br/>

## Running (docker)

``docker-compose up``

## After Running

``Access http://localhost:3000/``
