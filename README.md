# Contatooh

A simple contact manager implemented MEAN Stack.<br/>
This projects was based in "MEAN Book".

## Dependencies

``NodeJS``
``Bower``

## Persistence

``MongoDB``

## Running (locally)

``npm install``
``bower install``
``nodemon server.js``

## Running (docker)

``docker-compose up``

## After Running

``Access http://localhost:3000/``
